# 🛍️ FashionHub - E-Commerce Website

FashionHub is a full-stack E-Commerce web application built using ASP.NET Core.

## 🚀 Features
- Customer Registration & Login
- Admin Login & Dashboard
- Product Listing
- Add to Cart
- Order Placement
- Order Details
- Admin Panel Management

## 🛠️ Technologies Used
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- HTML, CSS, Bootstrap
